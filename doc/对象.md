# 对象

### 遍历属性

`for...in` 原型链、非symbol、可枚举

`...in...` 原型链、含symbol、含不可枚举

`Object.keys()` 自有、非Symbol、可枚举

`Object.getOwnPropertyNames()` 自有、非Symbol、含不可枚举

`Object.getOwnPropertySymbols()` 自有、仅Symbol、含不可枚举

`Reflect.ownKeys()` 自有、含Symbol、含不可枚举
